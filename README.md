# Kinetic Flow // Cyberpunk Portfolio

Benvenuto nel repository di **Kinetic Flow**, il portfolio professionale in stile **Cyberpunk** di **Manuela Musmeci** (Consulente Tecnica B2B e Laureanda in Ingegneria Informatica).

Questo progetto è un sito web **statico puro**, ottimizzato per essere distribuito ad altissima velocità su piattaforme come **Vercel** o **Netlify**.

---

## 🚀 Caratteristiche del Progetto
- **Aesthetic Cyberpunk**: Interfaccia dal forte impatto visivo con palette di colori neon (Cyan `#00f3ff` e Magenta `#ff00ea`), griglia digitale di sfondo ed effetti di luce dinamici.
- **Cursore Personalizzato**: Cursore interattivo che si adatta e si espande quando passa sopra elementi interattivi (link, card, pulsanti).
- **Single Page App (SPA) UX**: Navigazione fluida e veloce tramite animazioni Javascript per il cambio di sezione con dissolvenza hardware accelerata.
- **Statico & Leggero**: Realizzato interamente in HTML5, CSS3 e JavaScript vanilla. Zero framework pesanti, zero dipendenze esterne.

---

## 📂 Struttura del Progetto
La struttura del repository è estremamente pulita e ridotta al minimo essenziale per facilitare il deploy statico:

```text
├── index.html                      # Entry point e struttura delle sezioni (Home, Progetti, Contatti)
├── css/
│   └── style.css                   # Variabili di design, griglia cyberpunk e animazioni
├── js/
│   └── script.js                   # Navigazione delle sezioni e comportamento del cursore custom
├── README.md                       # Questa documentazione
└── .gitignore                      # Esclusioni per ambiente locale e di sviluppo
```

---

## ⚡ Sviluppo Locale
Puoi visualizzare il sito in locale in due modi semplici:

1. **Apertura Diretta**: Fai doppio clic sul file `index.html` per aprirlo direttamente nel tuo browser.
2. **Server Locale (Consigliato)**: Se usi VS Code, puoi utilizzare l'estensione **Live Server**, oppure eseguire da terminale:
   ```bash
   npx serve
   ```
   Quindi apri [http://localhost:3000](http://localhost:3000) nel browser.

---

## ☁️ Deploy su Vercel
Questo progetto è configurato per il deploy immediato a "zero configurazione" su **Vercel**:

1. Connettiti a [Vercel](https://vercel.com/) e collega il tuo account GitHub.
2. Clicca su **Add New** > **Project** e seleziona questo repository (`vercelapp`).
3. Vercel rileverà automaticamente il progetto come sito statico (HTML/CSS/JS).
4. Clicca su **Deploy**. In pochi secondi il tuo portfolio sarà online!
